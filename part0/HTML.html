Fixating HTML ans CSS formating

It's important to try yourself when learning. This applies not only for coding but for every lesson.

HTML and CSS are not programing language. HTML is a markup language and CSS is a style sheet language.

HTML Elements

  • Headings (h1 - h6)
  • Paragraphs (p)
  • Lists (ul, ol and dl)
  • Links (a)
  • Images (img)

CSS Selectors

h1 {  } Type selector
* {  } Universal selector
.box {  } Class selector
#unique { } id selector
a[title] {  } Attribute selector
p:first-child { } Pseudo-class selectors
p::first-line { } Pseudo-element selectors
article p Descendant combinator
article > p Child combinator
h1 + p Adjacent sibling combinator
h1 ~ p General sibling combinator
